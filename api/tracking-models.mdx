## Overview

The tracking models handle package verification, registration, and detailed tracking information. These models are spread across multiple files in `lib/features/scan/models/`.

---

## Verification Models

### VerifyTrackingNumberRequest

Request to verify if a tracking number is registered in the system.

#### Fields

<ParamField path="trackingNumber" type="String" required>

</ParamField>

#### Constructor


### VerifyTrackingNumberResponse

Response containing tracking number verification status.

#### Fields

<ResponseField name="isRegistered" type="bool" required>
  Indicates whether the tracking number has a registration
</ResponseField>

<ResponseField name="registrationDateTime" type="DateTime?">
  Date and time when the tracking was registered (if applicable)
</ResponseField>

#### Constructors

##### Default Constructor

```dart
const VerifyTrackingNumberResponse({
  required this.isRegistered,
  this.registrationDateTime,
});
```

##### empty()

Creates an empty response with `isRegistered = false`.

```dart
const VerifyTrackingNumberResponse.empty()
```

##### fromJson()

<Note>
  Supports both camelCase and PascalCase field names from the backend.
</Note>

```dart
factory VerifyTrackingNumberResponse.fromJson(Map<String, dynamic> json)
```

**Example:**

```dart
final response = VerifyTrackingNumberResponse.fromJson({
  'content': {
    'isRegistered': true,
    'registrationDateTime': '2026-03-05T14:30:00Z'
  }
});

if (response.isRegistered) {
  print('Registered on: ${response.registrationDateTime}');
}
```

#### Fields

<ParamField path="trackingNumber" type="String" required>
  The tracking number for the package
</ParamField>

<ParamField path="weight" type="double" required>
  Package weight in kilograms
</ParamField>

<ParamField path="width" type="double" required>
  Package width in centimeters
</ParamField>

<ParamField path="length" type="double" required>
  Package length in centimeters
</ParamField>

<ParamField path="height" type="double" required>
  Package height in centimeters
</ParamField>

<ParamField path="files" type="List<RequestEncodeFile>?">
  Optional list of image files to attach to the package
</ParamField>

#### Constructor

```dart
const RegisterPackageRequest({
  required this.trackingNumber,
  required this.weight,
  required this.width,
  required this.length,
  required this.height,
  this.files,
});
```

#### Methods

```dart
Map<String, dynamic> toJson()
```

<Info>
  The `files` array is only included in the JSON if it's not null and not empty.
</Info>

**Example:**

```dart
final request = RegisterPackageRequest(
  trackingNumber: 'TRACK123456789',
  weight: 2.5,
  width: 30.0,
  length: 40.0,
  height: 20.0,
  files: [
    RequestEncodeFile(
      encodeContent: base64Image1,
      fileExtension: 'jpg',
      fileName: 'front.jpg',
    ),
    RequestEncodeFile(
      encodeContent: base64Image2,
      fileExtension: 'jpg',
      fileName: 'back.jpg',
    ),
  ],
);
```

## Usage Examples

<CodeGroup>

```dart Complete Package Registration Flow
import 'package:logiscan/features/scan/models/register_models.dart';
import 'package:logiscan/features/scan/models/verify_tracking_models.dart';

Future<bool> registerNewPackage({
  required String trackingNumber,
  required double weight,
  required double width,
  required double length,
  required double height,
  List<RequestEncodeFile>? images,
}) async {
  // Step 1: Verify tracking number
  final verifyRequest = VerifyTrackingNumberRequest(
    trackingNumber: trackingNumber,
  );
  
  final verifyResponse = await httpService.post(
    '/verify-tracking',
    data: verifyRequest.toJson(),
  );
  
  final verification = VerifyTrackingNumberResponse.fromJson(
    verifyResponse.content,
  );
  
  if (verification.isRegistered) {
    print('Package already registered on ${verification.registrationDateTime}');
    return false;
  }
  
  // Step 2: Register package
  final registerRequest = RegisterPackageRequest(
    trackingNumber: trackingNumber,
    weight: weight,
    width: width,
    length: length,
    height: height,
    files: images,
  );
  
  final registerResponse = await httpService.post(
    '/register-package',
    data: registerRequest.toJson(),
  );
  
  return registerResponse.isSuccessful;
}
```

```dart Fetch Package Details
Future<GetPackageTrackingDetailsResponse> fetchPackageDetails(
  String trackingNumber,
) async {
  final response = await httpService.get(
    '/package-tracking-details',
    queryParameters: {'trackingNumber': trackingNumber},
  );
  
  final details = GetPackageTrackingDetailsResponse.fromJson(
    response.content,
  );
  
  // Display package information
  for (final item in details.items) {
    print('Tracking: ${item.trackingNumber}');
    print('Status: ${item.status}');
    print('Dimensions: ${item.length} x ${item.width} x ${item.height}');
    
    if (item.packageItem != null) {
      print('History:');
      for (final event in item.packageItem!.history) {
        print('  - ${event.event} at ${event.dateTime}');
      }
    }
  }
  
  // Display images
  print('Package has ${details.images.length} images');
  print('Upload allowed: ${details.allowUploadImages}');
  
  return details;
}
```

```dart Manage Package Images
Future<void> addImagesToPackage(
  int packageId,
  List<List<int>> imageBytesList,
) async {
  final files = imageBytesList.map((bytes) {
    return PackageImageEncodeFile(
      encodeContent: base64.encode(bytes),
      fileExtension: 'jpg',
      fileName: 'package_${DateTime.now().millisecondsSinceEpoch}.jpg',
    );
  }).toList();
  
  final request = RegisterPackageImagesByMobileRequest(
    packageId: packageId,
    files: files,
  );
  
  await httpService.post(
    '/register-package-images',
    data: request.toJson(),
  );
}

Future<void> removeImagesFromPackage(List<int> fileIds) async {
  final request = DeleteTrackingPackageImagesRequest(
    storedFileIds: fileIds,
  );
  
  await httpService.post(
    '/delete-package-images',
    data: request.toJson(),
  );
}
```
