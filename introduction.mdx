## What is LogiScan?

LogiScan is a comprehensive mobile logistics solution built with Flutter that combines QR code scanning, AI-powered package measurement, and real-time tracking capabilities. Designed for logistics professionals, LogiScan streamlines package management workflows with intelligent automation and seamless backend integration.

## Quick Start

Get started with LogiScan in minutes:

<Steps>
  <Step title="Install Dependencies">
    Clone the repository and install Flutter dependencies
    ```bash
    git clone https://github.com/JC-DEV-EC/LogiScan.git
    cd LogiScan
    flutter pub get
    ```
  </Step>
  <Step title="Configure Firebase">
    Set up Firebase for your project and add the configuration files
  </Step>
  <Step title="Run the App">
    Launch LogiScan on your device or emulator
    ```bash
    flutter run
    ```
  </Step>
</Steps>

## Architecture Overview

LogiScan is built with a clean, modular architecture:

- **Core Layer**: Shared services, HTTP client, storage, and error handling
- **Features Layer**: Domain-specific modules (authentication, scanning, tracking)
- **Presentation Layer**: Flutter UI components and state management with Provider
- **API Integration**: RESTful API client with automatic token refresh
