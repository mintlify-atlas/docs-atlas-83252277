## Overview

The `MeasurementService` class provides comprehensive functionality for handling package measurements, registering packages, verifying tracking numbers, managing package images, and retrieving package information. This is a core service for LogiScan's package scanning and tracking functionality.

**Location**: `lib/features/scan/services/measurement_service.dart`

## Constructor

```dart
MeasurementService(HttpService _http)
```

### Parameters

<ParamField path="_http" type="HttpService" required>
  The HTTP service instance for making authenticated API requests
</ParamField>

## Methods

### processMeasurementData

Processes measurement data from a scanned package file and extracts dimensional information.

```dart
Future<ApiResponse<ProcessMeasurementDataResponse>> processMeasurementData(
  ProcessMeasurementDataRequest request
) async
```

#### Parameters

<ParamField path="request" type="ProcessMeasurementDataRequest" required>
  Request containing the encoded file with measurement data
  
  ```dart
  ProcessMeasurementDataRequest(
    file: RequestEncodeFile(
      encodeContent: base64EncodedString,
      fileExtension: 'json',
      fileName: 'measurement_data.json',
    ),
  )
  ```
</ParamField>

#### Returns

<ResponseField name="ProcessMeasurementDataResponse" type="ApiResponse<ProcessMeasurementDataResponse>">
  Extracted measurement dimensions:
  - `weight` - Package weight in configured units
  - `width` - Package width
  - `length` - Package length
  - `height` - Package height
  
  All dimension values are nullable doubles.
</ResponseField>

#### Example Usage

```dart
final measurementService = MeasurementService(httpService);

final request = ProcessMeasurementDataRequest(
  file: RequestEncodeFile(
    encodeContent: base64.encode(measurementFileBytes),
    fileExtension: 'json',
    fileName: 'scan_${DateTime.now().millisecondsSinceEpoch}.json',
  ),
);

final response = await measurementService.processMeasurementData(request);

if (response.isSuccessful) {
  final data = response.content;
  print('Dimensions: ${data?.length}x${data?.width}x${data?.height}');
  print('Weight: ${data?.weight}');
}
```

### getPackageOwnerByImage

Identifies the package owner by analyzing a package image using image recognition.

```dart
Future<ApiResponse<GetPackageOwnerByImageResponse>> getPackageOwnerByImage(
  GetPackageOwnerByImageRequest request
) async
```

#### Parameters

<ParamField path="request" type="GetPackageOwnerByImageRequest" required>
  Request containing encoded image file for analysis
  
  ```dart
  GetPackageOwnerByImageRequest(
    file: RequestEncodeFile(
      encodeContent: base64ImageContent,
      fileExtension: 'jpg',
      fileName: 'package_label.jpg',
    ),
  )
  ```
</ParamField>

#### Returns

<ResponseField name="GetPackageOwnerByImageResponse" type="ApiResponse<GetPackageOwnerByImageResponse>">
  Package owner identification result:
  - `ownerPackage` - Enum indicating owner type (globalShipping, gbi, unknown)
  - `color` - Optional color identifier associated with the owner
</ResponseField>

#### Owner Types

```dart
enum OwnerType {
  globalShipping,
  gbi,
  unknown,
}
```

#### Example Usage

```dart
final request = GetPackageOwnerByImageRequest(
  file: RequestEncodeFile(
    encodeContent: base64.encode(imageBytes),
    fileExtension: 'jpg',
    fileName: 'label_scan.jpg',
  ),
);

final response = await measurementService.getPackageOwnerByImage(request);

if (response.isSuccessful) {
  final owner = response.content;
  switch (owner?.ownerPackage) {
    case OwnerType.globalShipping:
      print('Package belongs to Global Shipping');
      break;
    case OwnerType.gbi:
      print('Package belongs to GBI');
      break;
    case OwnerType.unknown:
      print('Package owner could not be determined');
      break;
  }
}
```

## Error Handling

All methods follow a consistent error handling pattern:

1. Wrap HTTP calls in try-catch blocks
2. Return `ApiResponse.error()` with appropriate error messages
3. Provide empty or default content objects on failure
4. Include error details from the API when available

```dart
if (!response.isSuccessful) {
  return ApiResponse.error(
    messageDetail: response.messageDetail,
    content: const GenericOperationResponse(userMessage: null),
  );
}
```

- `POST /process-measurement` - Process measurement data
- `POST /register-package` - Register new package
- `POST /verify-tracking-number` - Verify tracking number

- `POST /register-package-images` - Register package images
- `DELETE /delete-tracking-images` - Delete package images
